VOICE MAPPING FOR WINDOWS 11 - EXACT NAMES

Replace the speakText() function's voice matching section with:

```javascript
// Windows voice names - EXACT from Windows Narrator settings
const voiceMap = {
    'Microsoft Sonia': 'Microsoft Sonia - English (United Kingdom)',
    'Microsoft David': 'Microsoft David - English (United States)',
    'Microsoft Zira': 'Microsoft Zira - English (United States)',
    'Microsoft Mark': 'Microsoft Mark - English (United States)',
    'Google US English': 'Google US English',
    'Google UK English': 'Google UK English'
};

const actualVoiceName = voiceMap[voiceName] || voiceName;
let voice = voices.find(v => v.name === actualVoiceName);

// Fallback to partial match
if (!voice) voice = voices.find(v => v.name && v.name.includes(voiceName));
// Fallback to first available voice
if (!voice && voices.length > 0) voice = voices[0];

if (voice) {
    utterance.voice = voice;
    console.log('Using voice:', voice.name);
}
```
