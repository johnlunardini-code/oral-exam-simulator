## 🆘 If Something Goes Wrong

### Issue: App won't load
```
Try:
1. Hard refresh (Ctrl+F5 or Cmd+Shift+R)
2. Clear browser cache
3. Try different browser
4. Check Railway logs
```

### Issue: Feature not working
```
Try:
1. Check browser console (F12 > Console)
2. Look for red error messages
3. Clear browser cache
4. Try incognito/private mode
5. Check Railway logs
```

### Issue: Recording doesn't work
```
Try:
1. Check microphone permissions
2. Test microphone in system settings
3. Try different browser
4. Check Recording API support
5. Restart browser
```

### Issue: Meta-question not working
```
Try:
1. Type exact phrase: "what time is it?"
2. Check server logs
3. Try other meta-question phrases
4. Verify server response in Network tab (F12)
```
